# Skills

Agent skills for Claude Code, Cursor, Windsurf, and other AI coding agents.

## Skills

| Skill | Description | Install |
|-------|-------------|---------|
| conventional-commit | Angular conventional commit format | `npx skills add Darmikon/skills --skill conventional-commit` |
| pr-changelog | Generate PR changelogs from git diffs | `npx skills add Darmikon/skills --skill pr-changelog` |
| rethink | Step back and rethink systematically after 3+ failed attempts at any task | `npx skills add Darmikon/skills --skill rethink` |
| skill-publish | Create and publish skills to GitHub repos | `npx skills add Darmikon/skills --skill skill-publish` |

## Install All

```bash
npx skills add Darmikon/skills --skill conventional-commit --skill pr-changelog --skill rethink --skill skill-publish
```
